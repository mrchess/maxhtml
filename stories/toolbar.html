Toolbar

<mh-toolbar> is an elevated bar of tool buttons — a canvas/editor toolbar, a formatting bar. Children are <button type=button> tools; the active one carries aria-pressed="true", and an <hr> is a vertical divider between groups. Fully on-rails: it's just buttons + a native attribute (like mh-slots, but horizontal and elevated), reusable by any editor.

Example

Anatomy

The selected tool is aria-pressed="true" (filled) — a native, edit-proof signal, never a class. Groups are separated by <hr>. Use emoji as the icons (held as content, swappable for a future <mh-icon>); add aria-label since the glyph is the only text.

Usage

<mh-toolbar>
  <button type=button aria-pressed="true" aria-label="Select">↖</button>
  <button type=button aria-label="Sticky note">🗒</button>
  <hr>
  <button type=button aria-label="Image">🖼</button>
</mh-toolbar>
