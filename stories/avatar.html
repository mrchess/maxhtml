Avatar

A round container for a user's face. Holds an emoji (for now), initials, or an <img>. The icon is content, not CSS — so swapping the emoji for <mh-icon> later never touches layout.

Examples — emoji, initials, image

Usage

<mh-avatar>🧑</mh-avatar>                  <!-- emoji -->
<mh-avatar>AL</mh-avatar>                  <!-- initials -->
<mh-avatar><img src="..." alt=""></mh-avatar>  <!-- photo -->
