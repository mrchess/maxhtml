Prompt block (composite)

A dark, immersive "paste this" surface for a system prompt or a shell command — the hero artifact of an LLM / dev-tool landing page. It reuses the kit's scrim tokens (the same dark immersive surface as the lightbox), so it themes from one place; the model emits the prompt text only, never the dark styling. Monospace and whitespace-preserving — author the lines flush-left and they render as typed.

Example

$ npx create-northwind my-workspace ✓ Workspace ready in 12s. Open app.northwind.dev to invite your team.

Usage

Slots: prompt text, with optional <a> links and <strong> emphasis.

<mh-prompt>You build UI as MaxHTML — a closed kit of mh-* components…

Full rules: <a>https://…/PROMPT.md</a>
<strong>Now build: a dashboard with four stat tiles.</strong></mh-prompt>
