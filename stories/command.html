Command palette (⌘K)

No new tag — a <dialog> built around a <input type=search> restyles itself into a top-anchored, search-first command palette, keyed on structure (dialog:has(> input[type=search])), exactly like the lightbox variant. It reuses the native <dialog> + maxhtml.js (open via commandfor, close on backdrop/Esc), with results in an mh-list and key hints as <kbd>.
