Friends (recipe)

The friends-management screen from a social app: mh-app shell + mh-page + an mh-list of mh-items. Each row carries an avatar, identity, a primary action, and an overflow mh-menu for the long tail (poke, remove, block) — so "I might want more actions later" never touches the layout. Friend requests are the same item with Add friend / Decline instead.
