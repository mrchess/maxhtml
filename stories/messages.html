Messages (recipe)

A messaging app — mh-layout with a conversation-list pane (an mh-sidemenu holding a search box + a reused mh-list, active row = aria-current) and a conversation pane (an mh-navbar header + a sender-aligned mh-thread + an inline composer). The only new piece is the bubble thread (see Bubble thread); everything else is reuse. Zero styling emitted.
