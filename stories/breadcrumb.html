Breadcrumb

<mh-breadcrumb> is a horizontal trail of <a> crumbs; the current page is the last crumb, marked aria-current="page" (native). The / separators are inserted by CSS — never typed into the markup. Reused by product pages and docs.

Example

Usage

<mh-breadcrumb>
  <a href="/">Home</a>
  <a href="/shoes">Trail shoes</a>
  <a href="/shoes/tr2" aria-current="page">Trail Runner 2.0</a>
</mh-breadcrumb>
