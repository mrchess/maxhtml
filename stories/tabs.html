Tabs

<mh-tabs> is a horizontal in-page section-nav strip — settings sections, profile tabs, a product page's description/reviews. It's the content-level cousin of <mh-sidemenu>'s vertical nav: each tab is an <a> routing to its own URL, and the active one carries aria-current="page" (same model as <mh-submenu>). In-page panel switching without navigation is a JS graduation, like mh-menu.

Example

Usage

<mh-tabs>
  <a href="/settings" aria-current="page">Account</a>
  <a href="/settings/notifications">Notifications</a>
  <a href="/settings/billing">Billing</a>
</mh-tabs>
