Sidebar

A vertical navigation menu. Slots: a brand <header>, then a <nav> landmark holding one or more <mh-submenu> groups. Each group is a <header> label + <a> links. Mark the active link with aria-current="page".

Example — one group, three items

Example — two groups

Usage

<mh-sidemenu>
  <header>Acme</header>
  <nav>
    <mh-submenu>
      <header>Main</header>
      <a aria-current="page">Dashboard</a>
      <a>Projects</a>
      <a>Settings</a>
    </mh-submenu>
  </nav>
</mh-sidemenu>
