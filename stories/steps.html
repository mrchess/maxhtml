Steps + Progress

<mh-steps> is an ordered step indicator for a multi-step flow (checkout, onboarding, a wizard). Markup is an <ol> of <li> — ordered, so the numbers come free. Mark the current step aria-current="step"; done steps derive from structure (a step with a later current sibling is complete), so one marker drives done/current/upcoming with no per-step class. Determinate <progress> is the native bar, styled by the kit.

Example

Usage

<mh-steps>
  <ol>
    <li>Cart</li>
    <li aria-current="step">Shipping</li>   <!-- current; earlier = done -->
    <li>Payment</li>
  </ol>
</mh-steps>

<progress value="40" max="100"></progress>
