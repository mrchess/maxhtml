Pagination

<mh-pagination> is a one-row boxed pager for paged lists — search results, a data table, a blog index. Children are <a> page links (current = aria-current="page") plus prev/next links (disabled end = aria-disabled="true"); an ellipsis is a <small>. Distinct from mh-tabs (underline section nav) and mh-slots (wrapping chip grid).

Example

Usage

<mh-pagination>
  <a href="?p=0" aria-disabled="true">← Prev</a>
  <a href="?p=1" aria-current="page">1</a>
  <a href="?p=2">2</a>
  <small>…</small>
  <a href="?p=9">9</a>
  <a href="?p=2">Next →</a>
</mh-pagination>
