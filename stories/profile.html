Profile (recipe)

A user profile: mh-app shell + an mh-profile header (a big avatar — same mh-avatar, slot-scoped larger — plus name, meta, and actions) + an mh-card "About" + an mh-list of mutual friends. Message and Remove friend open native <dialog>s.
