Icon

<mh-icon name="…"> is a mask-based SVG icon. The artwork (Lucide, ISC) lives in maxhtml.css as a URL-encoded mask — the model emits only the tag and the name, never the SVG, so an icon costs about the same as an emoji while being themeable. background is currentColor, so an icon takes the colour of its surrounding text and is fully recolorable; it's sized 1em, so it scales with the type around it.

The set

Inherits text colour & size

Because it's currentColor at 1em, an icon just works inline in a button, a link, a heading, or a badge — no sizing or colour to emit:

Usage

<button><mh-icon name="download"></mh-icon> Download</button>
<mh-badge><mh-icon name="message"></mh-icon> 18</mh-badge>
<mh-avatar><mh-icon name="user"></mh-icon></mh-avatar>   <!-- anonymous avatar -->

Closed set — adding an icon means adding one mask rule in maxhtml.css (and logging it). Emoji are reserved for genuine user content (a chat message, a reaction), never UI chrome. Severity colour rides the separate tone attribute (see Badge / Alert).
