Contact record (recipe)

A record page — mh-profile header + mh-stat row + a property panel and an activity mh-list, plus a dense multi-column edit form in a <dialog>. Two patterns worth noting, both pure reuse:

Property panel = a plain <table> whose row label is a <th scope="row">. The kit detects the row-header structurally (a <th> in <tbody>) and renders the label column muted + narrow — no new component.

Dense form = a <form> whose fields sit in an mh-grid of <label>s, so they flow into 2–3 columns; the dialog widens itself because it contains a field grid.
