App shell

The full dashboard shell: a top <mh-navbar>, a left <mh-sidemenu>, and a <main> content region. <mh-app> is a CSS grid that places the three as direct children — no wrappers, no classes.

Anatomy

Slots: <mh-navbar> spans the top, <mh-sidemenu> is the left column, <main> fills the rest. See it live in the Dashboard recipe.

Usage

<mh-app>
  <mh-navbar>...</mh-navbar>
  <mh-sidemenu>...</mh-sidemenu>
  <main>
    <mh-page>...</mh-page>
  </main>
</mh-app>
