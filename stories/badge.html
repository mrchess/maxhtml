Badge

<mh-badge> is a small neutral status pill — a building block for table cells, record headers, and board cards. It's an atom, not where MaxHTML competes: it exists so composites have a pill to drop in.

Example

Colour rides on an emoji, not a class

MaxHTML deliberately has no colour-per-variant (a danger badge, a success badge) — that would need a class or data-* hook the kit forbids. Instead the badge is one neutral pill and the status colour is carried by a leading emoji dot as content (🟢 / 🟡 / 🟠 / 🔴), the same convention the calendar uses for 🟢 Personal. Honest limitation: you get an at-a-glance dot, not a fully colour-filled pill background.

Usage

<mh-badge>🟢 Customer</mh-badge>
<td>…<mh-badge>🔴 Churned</mh-badge>…</td>     <!-- in a table cell -->
<h1>Ada Lovelace <mh-badge>🟢 Customer</mh-badge></h1>  <!-- in a record header -->
