Typography & prose

For long-form docs and articles the kit styles the native prose elements directly — headings <h1>–<h4>, <p>, <ul>/<ol>, <blockquote>, inline <code>, <pre> code blocks, and <kbd> for keys. Atoms, no new tags. Pair with mh-alert for callouts and <details> for FAQ.
