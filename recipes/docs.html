The Acme SDK gives you typed access to the Acme API in a few lines. This guide covers installing it and making your first authenticated request.

Requirements

  • Node 18 or later (LTS recommended)
  • An Acme account and an API key
  • A package manager — npm, pnpm, or yarn
Note. Use the current LTS release of Node for production deployments.

Install the SDK

Install from npm, then import the client:

npm install @acme/sdk

Verify the install with npx acme --version.

Tip: pin the SDK version in CI so builds are reproducible.

Configuration

The client accepts these options:

OptionTypeDescription
apiKeystringYour secret API key. Required.
baseUrlstringOverride the API host. Defaults to api.acme.com.
timeoutnumberRequest timeout in ms. Defaults to 10000.

FAQ

Where do I find my API key?

In the dashboard under Settings → API keys. Keep it secret; rotate it if exposed.

Does the SDK work in the browser?

Yes, but never ship your secret key to the browser — proxy requests through your server.
